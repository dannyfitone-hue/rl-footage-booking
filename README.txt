RL FOOTAGE Stripe Checkout Booking Page

This package keeps the original landing page design and adds a Stripe Checkout backend endpoint.

Run locally:
1. Install Node.js
2. Open Terminal in this folder
3. npm install
4. STRIPE_SECRET_KEY=sk_test_YOUR_KEY DOMAIN=http://localhost:3000 npm start
5. Open http://localhost:3000

For live deployment, set:
STRIPE_SECRET_KEY=sk_live_YOUR_KEY
DOMAIN=https://yourdomain.com

Do not put your Stripe secret key inside index.html.
